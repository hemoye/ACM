问题描述
我们划分的HZNU的校园为N * m的网格。从下面的图片中，你可以看到绿色的网格代表对楼宇。由于大小的HZNU校园，每个网格的颜色，你应该看看有多少人绿色网格，在N * M的网格。


 

输入
标准输入将包含多个测试案例。输入的第一行是一个整数T，它是多的测试用例。T检验的情况下，遵循
每个测试案例的第一行包含两个整数n和m（1 <= N，M <= 100），在校园的大小。然后，按照n行，每 ??行包含m个整数。在第i行的第j个整数是该网格的颜色为白色，0表示，而1表示绿色。

 

产量
结果应直接输出到标准输出。对于每一种情况下，输出一个整数T，N * M大小校园总绿化网格。
 

样例输入
2 
2 2 
1 1 
0 0 
3 3 
1 0 1 
0 0 1 
1 1 0

 

样本输出
2 
5
