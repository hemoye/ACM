最低位

时间限制：2000/1000ms（JAVA /其他）内存限制：65536/32768K（JAVA /其他）
的总提交（）：0接受的意见书（S）：0
问题描述
给定一个正整数（1 <= A <= 100），输出的最低位A. 例如，A = 26，我们可以写一个以二进制的形式为11010，最低位是10，所以输出应该是2。另一个例子是这样的：A = 88，我们可以写一个以二进制形式为1011000的最低位，一个是1000，所以输出应为8。




 

输入
输入的每一行只包含一个整数（1 <= A <= 100）。含“0”的A线表示输入的结尾，这条线是没有输入数据的一部分。
 

产量
对于每一个输入，输出线只包含的最低位。
 

样例输入
26 
88 
0
 

样本输出
2 
8
