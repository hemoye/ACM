Problem C: 找小强

Time Limit: 2 Sec  Memory Limit: 64 MB
Submit: 165  Solved: 28
[Submit][Status][Web Board]
Description
软件学院最近组织了大一年级的学生在机房搞C语言进阶，来了许多同学（看来大家真是爱学习）。2月的天，张家界的小雨，天气阴阴冷冷滴.>_<...于是有许多同学在AC-ing带了干粮补充energy。不幸的是，有些同学不小心洒落了干粮碎屑...
在漆黑的晚上，机房，一只只不明生物体正在啃食...没错，那就是自然界的清道夫之一---小强。由于小强啃食过的地方可能会留下不干净的东西，为了同学们的健康成长，小丹决定单枪匹马驱逐小强。路见不平，当拔刀相助！作为一个友爱的师弟，你决定帮他制作一个“小强探测器”，能够自动探测以当前位置为中心一定范围的小强数目。现在你需要解决一个难题，假设你已经知道小强在机房各个位置的具体数量，给出小丹的当前位置，输出3X3范围内的小强数目。
Input

第一行是一个整数T(T<50)，表示有T组输入数据
每组输入数据第一行是两个整数rows和cols，分别表示机房的长度和宽度
然后是rows行，每行cols个整数，表示各个位置的小强数目
接着多个输入小丹所在的行列row, col,以( -1, -1 )结束

Output

对于每个XX所在的位置，输出只有一行，表示探测到的小强数目

Sample Input

1
3 3
1 2 1
2 0 2
1 2 1
1 1
-1 -1
Sample Output

12
